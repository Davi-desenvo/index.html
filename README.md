# NOIR — Moda Casual Premium

Site institucional de uma marca fictícia de moda casual premium, desenvolvido com foco em design elegante, minimalista e moderno.  
O projeto simula a presença digital de uma marca voltada a um público de alto padrão aquisitivo, priorizando clareza, estética e posicionamento.

## Visão Geral

A NOIR nasce da ideia de que menos é poder.  
O site apresenta uma identidade visual escura, tipografia limpa e estrutura institucional sólida, seguindo padrões utilizados por marcas premium e startups modernas.

O objetivo do projeto é demonstrar boas práticas em:
- Estruturação de páginas web
- Design limpo e sofisticado
- Organização de código HTML e CSS
- Publicação de sites estáticos com GitHub Pages

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- Google Fonts (Inter)  
- GitHub Pages (hospedagem)

## Estrutura do Projeto

noir-store/
└── index.html
O projeto foi mantido propositalmente simples, utilizando um único arquivo HTML com CSS embutido, facilitando a leitura, manutenção e publicação.

## Funcionalidades

- Layout responsivo
- Navegação institucional
- Seções de coleção, diferenciais, sobre a marca e contato
- Design dark premium
- Estrutura pronta para expansão futura

## Publicação

O site está hospedado via **GitHub Pages**.

Para publicar:
1. Suba o arquivo `index.html` na raiz do repositório
2. Vá em **Configurações → Pages**
3. Selecione o branch principal (`main` ou `master`)
4. Escolha a pasta `/(root)`
5. Salve e aguarde a geração do link
